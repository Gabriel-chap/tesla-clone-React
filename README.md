# Tesla Website redesign
Demo: https://tesla-redesign-gc.netlify.app/
This project is a simplified front end redesign of the Tesla website. It was created with React.

Tools used:

    Webpack
    Redux & React
    Media queries
    React Reveal
    Material UI - Material Icons
    Styled Components
